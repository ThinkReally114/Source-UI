## 技術棧

- **前端**: HTML5, CSS3, JavaScript (原生)
- **樣式**: CSS Variables, Flexbox, Grid
- **動畫**: CSS Transitions, CSS Animations
- **圖標**: SVG

- ## 本地運行

```bash
# 克隆倉庫
git clone https://github.com/ThinkReally114/Source-UI.git

# 進入目錄
cd Source-UI

# 打開文件（任選其一）
# 方式1: 直接用瀏覽器打開
open index.html

# 方式2: 使用本地服務器
npx serve .
# 或者
python -m http.server 8080
```

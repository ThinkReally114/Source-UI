## 技术栈

- **前端**: HTML5, CSS3, JavaScript (原生)
- **样式**: CSS Variables, Flexbox, Grid
- **动画**: CSS Transitions, CSS Animations
- **图标**: SVG 图标

- ## 本地运行

```bash
# 克隆仓库
git clone https://github.com/ThinkReally114/Source-UI.git

# 进入目录
cd Source-UI

# 打开文件（任选其一）
# 方式1: 直接用浏览器打开
open index.html

# 方式2: 使用本地服务器
npx serve .
# 或
python -m http.server 8080
```
